## Before You Install i3wm:
 ## UPDATE REPO 

  Debian/Ubuntu : 
      sudo apt update  
  
 Arch Linux : 
sudo pacman -Syu 

 Fedora : 
sudo dnf update 

## INSTALL git 

 Debian/Ubuntu : 
    sudo apt install git 

Arch Linux : 
         sudo pacman -S git 
         
 Fedora : 
    sudo dnf install git
